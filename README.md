# Dylan Robinson — Portfolio

A static, single-page portfolio positioning Dylan as a backend-first software
engineer with experience across distributed systems, cloud services, full-stack
web development, applied NLP research, education, and professional delivery.

## Local preview

Serve the repository root with any static file server:

```sh
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Structure

- `index.html` — semantic page content and metadata
- `styles.css` — responsive visual system, diagrams, and motion preferences
- `script.js` — accessible project tabs, section tracking, and reveal behavior
- `assets/` — favicon, profile image, social preview, and résumé

## Accessibility

The site includes a skip link, keyboard-operable project tabs, visible focus
states, reduced-motion support, semantic headings, diagram descriptions, and a
readable no-JavaScript fallback.

## Deployment

The site is deployed through GitHub Pages at
